Issues:

  1. Text in blender won't been exported as GLTF meshes
    This means blender can't export all it's resource into GLTF. And text objects in blender won't been loaded.
  2. Node's names are sanitized
    In three.module.js:
    	/**
	 * Replaces spaces with underscores and removes unsupported characters from
	 * node names, to ensure compatibility with parseTrackName().
	 *
	 * @param {string} name Node name to be sanitized.
	 * @return {string}
	 */
	sanitizeNodeName: function ( name ) {
		return name.replace( /\s/g, '_' ).replace( _reservedRe, '' );
	},
    This means node names like "route66 panel" can only been usable with name "route66_panel" - x-visual also have to "sanitize" it.
